Currency

The project aims to develop a Java Script based system whose function is to calculate and convert foreign currency rates automatically and accurately. The system will provide an applied solution for users who are required to make conversions between different currencies and be updated on the current exchange rates, the system will be based on modern Java Script technologies, including the use of external databases to receive currency rates in real time.
