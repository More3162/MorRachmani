X & O Game

This project is a Java Script application that develops the classic game X Circle. The game offers two players to play in turn on a board measuring 3 x 3. Each player chooses between an X or a circle and places the mark in the cells on the board according to his turn. The system checks if one of the players has achieved a sequence of three marks on the horizon, vertically or diagonally, and announces the winner. If the board fills up without anyone winning, the game ends in a draw.
