Page Builder

The Screen Builder project is a JavaScript application that allows users to create and customize HTML elements on a page of their choice. Users can choose to add elements to the page. They can then adjust the properties of the selected element, including height, width, background color, font type, text, font size, and font color. Users make the adjustments through a simple and friendly interface, and the selected specification is updated immediately on the page. The project provides a flexible and easy way to create custom designs while maintaining intuitiveness and simplicity.
