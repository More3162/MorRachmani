Weather App

This project is a Java Script application that allows the user to enter the name of a city in the world and receive up-to-date data on it. After entering the name of the city, the page displays data such as current time, temperature, outdoor visibility, humidity and wind speed. In addition, the page also indicates in which country the city is located. The data is retrieved through a connection to an external software interface that provides information about the weather and current time. The interface is user-friendly, allows receiving information quickly and easily, and provides a complete picture of the state of the selected city in real time.
